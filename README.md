# Smart Health Insurance Claim Processing and Fraud Detection System 🏥🤖

This repository contains the web prototype for the **Smart Health Insurance Claim Processing and Fraud Detection System**, developed as a **Complex Computing Problem (CCP)**. This system automates the health insurance claim lifecycle, validates policy coverage, and calculates fraud risk scores to prevent financial leakages.

### 🌟 Key Features of the Prototype
* **Online Claim Submission:** Seamless interface for hospitals to upload treatment records and claims.
* **Automated Rule Validation:** Checks policy boundaries and hospital eligibility dynamically.
* **Fraud Risk Scoring:** Provides real-time automated fraud detection and risk tiering.
* **Role-Based Portals:** Dedicated dashboards for Hospitals, Policyholders, and Insurance Review Officers.
* **Live Claim Tracking:** Transparency for policyholders to monitor claim approval status in real-time.
